# Team Jonathan und Keanu

## Places

Wir bauen eine Webanwendung, mit der Nutzer Orte, Sehenswürdigkeiten oder besondere Plätze, die sie besucht haben, mit der Community teilen können.


Jeder Nutzer kann:

einen **Ort** hinzufügen,
ein **Foto** hochladen,
eine **Bewertung** und einen **Kommentar** hinterlassen.

Andere Nutzer können diese Orte über eine Google Maps Integration entdecken.
